Projects

Data engineering and visualization projects I have led

Weather Data Visualization for Cities Nationwide (30-Day Period)
Project Lead
Sophomore Internship Project
Built a data analysis and visualization pipeline based on the pyecharts platform, covering weather conditions across different regions of China.
  • Collected weather data for popular cities using XPath-based HTML parsing as the data acquisition layer.
  • Processed data with Python Pandas and statistical formulas; derived 30-day maximum average temperatures for major cities, capturing date and temperature features reliably even under extreme weather conditions.
  • Implemented consistent, complete multi-region visualizations using pyecharts chart composition and image stitching strategies.
Python Pandas pyecharts XPath Data Visualization
Box Office Data Analysis & Visualization Based on Hive
Project Lead
Junior Internship Project
Performed end-to-end data analysis and visualization on Douban movie information using the Hive big data platform.
  • Built a Scrapy-based crawling framework to collect Douban movie data; designed a low-cost storage pipeline combining local storage and Pipe storage, with MySQL as the persistence layer.
  • Constructed a layered Hive data warehouse: ODS for structured data ingestion, DWD for pre-processing, DWS for price and basic-info analysis, and ADS for wide-table generation covering box office revenue and review counts.
  • Delivered interactive, exploratory multi-screen visualizations on the data analytics platform.
Hive Scrapy MySQL Big Data Data Warehouse ODS / DWD / DWS / ADS
